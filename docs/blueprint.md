# **App Name**: Impulso Fitness

## Core Features:

- User Authentication & Registration: Enable secure user login and registration, including a 'Sign in with Google' option, and a distinct sign-up flow with clean input fields that highlight on focus.
- Service & Class Catalog: Allow users to browse available gym services, class schedules, and membership plans with detailed descriptions displayed in clean, icon-supported blocks.
- Lead Capture Form: A prominent single-line form in the hero section for collecting user emails to drive engagement and future sign-ups.
- AI Workout Assistant Tool: A generative AI tool that provides basic workout suggestions or prompts for creating a personalized fitness plan based on user-inputted goals (e.g., strength, cardio, weight loss).
- User Dashboard Preview: A static display illustrating a personalized 'Next Class' card and 'Weekly Progress' graph to entice users about the personalized post-login experience.
- Testimonials & Social Proof Display: Showcase success stories with minimalist cards featuring circular member photos, 5-star ratings, and a section for allied brand logos or certifications in grayscale.

## Style Guidelines:

- Primary action color: 'Action Purple' (#635BFF), used for main CTAs, active states, and interactive elements for a vibrant look.
- Main background color: Pure White (#FFFFFF), ensuring a clean and minimalist canvas.
- Secondary background color: Soft Bluish Gray (#F7F9FC), subtly differentiating sections and adding depth without visual clutter.
- Heading text color: Very Dark Gray (#1A1A1A) for high contrast and readability.
- Body text color: Muted Gray (#4F5B76) for comfortable reading.
- Border color: Light Gray (#E6E8EB) for input fields and subtle separators.
- Font family: 'Inter' (sans-serif) for all text. Headlines (e.g., H1) use font-weight: 800, size: 3rem, and letter-spacing: -0.02em. Body text uses font-weight: 400, size: 1.125rem.
- Lineal/Outline icons with 2px thickness, matching the heading text color for a modern and precise visual identity.
- A 12-column grid layout with 32px gutters and a maximum container width of 1280px, following an 8px base spacing system for consistent and generous whitespace.
- User interface components feature generous rounded borders: 8px for buttons, 12px for cards, and 6px for input fields.
- Input fields have a white background, light gray borders, and display a soft purple ring on focus, emphasizing interaction.
- Subtle hover effects and focus transitions on interactive elements like buttons and input fields to provide engaging user feedback.
